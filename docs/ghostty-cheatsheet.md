# Ghostty Cheat Sheet

## Passthrough to tmux/nvim

| Key | Action |
|-----|--------|
| `Ctrl+H/J/K/L` | Passed through to tmux/nvim for pane navigation |

## Window Management

| Key | Action |
|-----|--------|
| `Cmd+N` | New window |
| `Cmd+T` | New tab |
| `Cmd+W` | Close tab/pane |
| `Cmd+Shift+W` | Close window |

## Tabs

| Key | Action |
|-----|--------|
| `Cmd+1-9` | Go to tab 1-9 |
| `Cmd+Shift+[` | Previous tab |
| `Cmd+Shift+]` | Next tab |

## Text

| Key | Action |
|-----|--------|
| `Cmd+C` | Copy |
| `Cmd+V` | Paste |
| `Cmd+A` | Select all |
| `Cmd+K` | Clear screen |

## Font Size

| Key | Action |
|-----|--------|
| `Cmd++` | Increase font |
| `Cmd+-` | Decrease font |
| `Cmd+0` | Reset font size |

## Config

| Key | Action |
|-----|--------|
| `Cmd+Shift+,` | Reload config |

## Config File Location

```
~/Library/Application Support/com.mitchellh.ghostty/config
```

## Quick Settings

Edit config file:

```bash
# Font
font-family = Berkeley Mono
font-size = 13

# Opacity
background-opacity = 0.8

# Padding
window-padding-x = 8
window-padding-y = 8
```
